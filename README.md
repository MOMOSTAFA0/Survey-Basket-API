# Survey Basket API
Survey Basket API is a production-style backend system built using ASP.NET Core Web API. It provides a complete solution for creating, managing, and analyzing surveys.

## 📌 Purpose
This project was built to apply backend engineering concepts and simulate real-world production scenarios including authentication, scalability, and maintainable architecture.

## 🚀 Features
- JWT Authentication & Refresh Tokens
- Role-based Authorization (Roles & Permissions)
- Survey Management (Create, Update, Publish, Delete)
- Questions & Voting System
- Results & Analytics (votes per day / per question)
- Account Management (Email confirmation, Password reset)
- Pagination, Filtering & Sorting

## 🏗️ Architecture
- Clean / Layered Architecture
- Separation of Concerns
- DTOs & Mapping
- Modular structure (Polls, Questions, Votes, Results, Users, Roles)

## 🛠️ Tech Stack

## ▶️ Getting Started
1. Clone the repository  
2. Update your connection string in `appsettings.json`  
3. Run database migrations  
4. Run the project
